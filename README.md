# Last Town Before Utah — Dirty Edge · Fruita, CO

A static editorial write-up of the Dirty Edge gravel race in Fruita, Colorado, authored by M. Ajayi for Melanated Lovelies. Built as a single `index.html` file with embedded CSS and no dependencies beyond two Google Fonts; drop your media files into `/images/` (three `.jpg` photos) and `/video/` (`hero.mp4`), push to GitHub, and enable Pages under **Settings → Pages → Branch: main / root**.
